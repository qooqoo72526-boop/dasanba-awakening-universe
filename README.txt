
# 大三巴覺醒宇宙 v8.9 Cosmic Final (Vercel + GitHub)

- 上傳整個資料夾到 GitHub（不要只上傳 zip 內層）
- Vercel 連接此 repo，一鍵 Deploy
- 在 Vercel → Settings → Environment Variables 新增：
  - `OPENAI_API_KEY` = 你的金鑰
- 把你的素材放進 `/assets/`：
  - trio.mp4（可選）
  - trio.webp ~ trio6.webp
  - ajin.webp / migou.webp / gungun.webp
  - ajin_5s.mp4 / migou_5s.mp4 / gungun_5s.mp4
  - ajin_clear.webp / migou_clear.webp / gungun_clear.webp（若要在結果處動畫嘴型可用）
  - bgm_universe.mp3（首頁） / room_ambient.mp3（星艙） / mirror_reflect.mp3（照妖鏡）
